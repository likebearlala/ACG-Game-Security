# 心動戀愛模擬器

一款繁體中文互動式戀愛劇情遊戲，也是 ACG-GAME 的兒童安全版。故事從單身生活開始，逐步進入曖昧、約會、告白與正常戀愛，不再使用外遇、欺瞞或危險關係作為核心衝突。

## 遊玩方式

直接開啟：

```text
index.html
```

建議用桌面瀏覽器遊玩，手機瀏覽器也支援響應式版面。

## 特色

- 五步開局設定：身份、目前關係、攻略角色、故事風格、情感濃度
- 從單身到曖昧，再到穩定戀愛的 12 週進展
- 多個可攻略戀愛角色：穩定型與心動型角色會以不同節奏靠近玩家
- 攻略過程可包含牽手、擁抱、親吻、約會與非露骨的過夜陪伴
- 數值變動彈窗，依角色與玩家狀態顯示頭像
- 自訂行動輸入框，可自由嘗試傳訊息、邀約、告白或保持界線
- 本機存檔與讀檔
- 多結局專頁，依選擇顯示戀愛、友好收束或自我成長路線

## 內容界線

本版本聚焦健康、尊重與非露骨戀愛。角色設定為成年人；故事不包含未成年人性化、外遇、操控、報復、露骨色情或暴力脅迫。
