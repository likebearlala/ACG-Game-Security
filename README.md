# 危險曖昧模擬器

一款繁體中文互動式戀愛劇情遊戲。故事保留欺瞞、試探、報復、關係失控與多線曖昧的戲劇張力，但排除「已婚」、「已有現任」、「既定伴侶」作為前提。

## 遊玩方式

直接開啟：

```text
index.html
```

這是純前端專案，不需要安裝依賴或啟動伺服器。

## 特色

- 七步開局設定：身份、自己的性別、目前關係、攻略對象性別、危險對象、故事風格、情感濃度
- 從單身、剛認識或曖昧狀態開始，不包含已婚或已有伴侶設定
- 多個可攻略戀愛角色：可選攻略對象性別，主要攻略角色與危險對象會以不同方式靠近玩家
- 保留欺瞞、IG 可見範圍、深夜訊息、報復與關係失控等危險關係框架
- 攻略過程可包含肢體接觸、親吻、過夜等非露骨親密橋段
- 數值變動彈窗，依角色與玩家狀態顯示頭像
- 本機存檔與讀檔
- 多結局專頁，包含坦白、抽身、報復、翻車與雙線崩盤等走向

## 內容界線

角色設定為成年人。此版本可以描寫欺瞞、報復和情感傷害，但不使用已婚、現任伴侶、未成年人性化或露骨色情作為內容。
