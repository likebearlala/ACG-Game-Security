# 危險關係模擬器

一款繁體中文互動式戀愛劇情遊戲。故事核心不是單純戀愛，而是忠誠、欲望、謊言與選擇。玩家在已有伴侶的狀態下，面對新的情感誘惑；每一次選擇都會改變數值、關係階段與結局走向。

## 遊玩方式

直接用瀏覽器開啟：

```text
index.html
```

這是純前端專案，不需要安裝依賴或啟動伺服器。

## 主要功能

- 五步開局設定：身份、目前關係、危險對象、故事風格、情感濃度
- 每週 5 個行動點
- 現任、危險對象、玩家三組數值系統
- LINE / IG 情境事件
- IG 摯友、限時動態、定位、謊言與翻車風險
- 數值變動彈窗，依現任、危險對象、玩家分組顯示頭像
- 存檔、讀檔、重新開始新周目
- 最多第 12 週進入大結局
- 大結局專頁，依結局顯示現任、危險對象或玩家頭像

## 結局規則

遊戲最晚會在第 12 週進入大結局。若數值提前爆發，也可能提早結束。

目前包含：

- 主動坦白
- 悄然抽身
- 危險對象失控
- 徹底翻車
- 雙線崩盤

本遊戲包含出軌、欺瞞、情感傷害、冷戰與關係崩裂等成人議題，僅供成年玩家進行情感敘事與心理探討。
